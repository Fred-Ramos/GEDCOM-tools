# GEDCOM tools (.ftz/.ftt file converter)

This repository converts .ftz family tree files to the general standard format GEDCOM, which can easily be used for printing and in other programs.

## Usage

Clone the repository, make sure **python** is installed and then place all **.ftz** you which to convert in the same directory as **main.py**. Then simply open the directory location in the terminal and run:

   ```bash
   python main.py
   ```
